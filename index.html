4. Apa itu rest API, SQL, HTTP, UI framework

-)rest API

adalah standar pembuatan atau arsitektur yang menggunakan protokol http secara umum yang sering di gunakan seperti (GET,POST,DELETE dan lainya)

-)SQL atau (Structured Query Language)

adalah sebuah bahasa standar yang di gunakan dalam manajemen basidata relasional yang saat ini hampir digunakaan semua server basisdata

-)HTTP atau (Hypertext Transfer Protocol)

adalah yang mengatur komunikasi antara client dan server dalam bentuk hal ini client adalah browser yang menampilkan konten WEB

-)UI Framework atau (User Interface Framework)

adalah sebuah tampilan User atau halaman depan WEB yang diatur agar lebih menarik dan informasi mudah untuk digunakan
